1.- • Editar el archivo "gameinfo.txt" que se encuentra en la carpeta "left4dead2":
C:\Program Files (x86)\Steam\steamapps\common\Left 4 Dead 2\left4dead2\
Agregar el nombre de la carpeta que crearas en el juego
Encuentra esta parte del archivo Search Paths:

SearchPaths
{
	Game				MisAddons
	Game				update
	Game				left4dead2_dlc3
	Game				left4dead2_dlc2
	Game				left4dead2_dlc1
	Game				|gameinfo_path|.
	Game				hl2
}

Agregas y guardas los cambios en el archivo.

C:\Program Files (x86)\Steam\steamapps\common\Left 4 Dead 2
